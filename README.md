# fanc-api

# 概要
 [PitaScho](https://pitascho.com/)の管理サービス[fanc](https://fancapp.com/login)はSPA構成で作成されていてそのバックエンドAPIのリポジトリ

# 使用技術/ツール概要

- Go
- echo
- gorm
- SendGridAPI
- SlackAPI
- jicoo(日程予約/決済)
- GitHub Actions
- Docker
